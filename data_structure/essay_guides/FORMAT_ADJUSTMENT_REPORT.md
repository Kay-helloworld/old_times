## 已完成的調整

### 1. ✅ 移除 Code Block 標記
- **問題**：原檔案中每道題目都被 ` ```text ... ``` ` 包裹
- **調整**：移除所有 code block 標記，題目內容直接以純文字形式呈現
- **結果**：格式與參考檔案一致

### 2. ✅ 整理空白行
- **問題**：文件中有過多的連續空白行
- **調整**：將 3 個以上連續換行統一調整為 2 個空行
- **結果**：文件結構更清晰

### 3. ✅ 修正題號換行
- **問題**：部分題號（一、二、三、...）被合併到前一行
- **調整**：在題號前自動加入換行
- **結果**：題號獨立成行，更容易閱讀

### 4. ✅ 修正表格格式
- **問題**：相鄰矩陣等表格被合併成單行
- **調整**：在表格行標識符前加入換行
- **結果**：表格結構更清晰

### 5. ✅ 修正程式碼格式
- **問題**：程式碼關鍵字被合併到同一行
- **調整**：在 `if`, `return`, `for`, `while` 等關鍵字前加入換行
- **結果**：程式碼可讀性提升

由於原始 PDF 轉文字時就存在問題，以下情況可能仍需手動調整：

1. **長句換行**：某些句子在 PDF 轉換時被拆成多行，部分合併邏輯可能不完美
2. **特殊符號**：數學符號、圖表說明等可能格式不統一
3. **代號與頁次**：題目末尾的「代號」「頁次」資訊可能需要進一步整理

### 建議後續處理
