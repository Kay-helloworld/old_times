#### 一、(資料結構)

```text
一、假設有一陣列 A 儲存了 n 筆資料，我們想要在陣列 A 中搜尋是否存在一筆資料
x。如果 x 存在陣列 A 中，則回傳 x 在陣列 A 中的索引值（index）；如果 x 不存
在陣列 A 中，則回傳-1。
請使用 C、C++、Java 或 Python 程式語言，寫出一個使用迴圈（loop）的程式
來執行上述搜尋工作。（10 分）
請使用 C、C++、Java 或 Python 程式語言，寫出一個使用遞迴（recursion）的
程式來執行上述搜尋工作。（10 分）
```

#### 二、(資料結構)
**關鍵字**: 節點, 鍵值, 說明, 相同, 集合, 比較, 差異, 結構, AVL Tree, Heap, 堆積, 2-4 Tree, 樹, C

```text
二、假設我們有相同的資料集合（鍵值分別為 2, 4, 5, 7, 8, 9, 15, 20, 30, 40），請分別
說明下列各資料結構中節點鍵值的關係，並畫出圖形以呈現資料儲存的結果。
```

```text
三、假設一份文件包含 6 個字元，其出現頻率分別為：a: 40k, b: 20k, c: 15k, d: 10k,
e: 8k, f: 7k。
請畫出 Huffman coding 建立的樹。（10 分）
計算編碼後的總位元數（total bits）。（5 分）
```

```text
拜訪順序，並畫出佇列（Queue）或堆疊（Stack）在執行過程中的內容變化。

深度優先搜尋（Depth-First Search, DFS）。（10 分）
（圖形略：A 連接 B, D, E；B 連接 A, C, E；C 連接 B, F；D 連接 A, E；E 連接 A, B, D, F；F 連接 C, E）
```
