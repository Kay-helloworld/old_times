## 📋 關於無法辨識字符的說明

### 問題：`m□n` 顯示
用戶反映看到類似 `m□n` 的顯示問題。

### 原因分析
經檢查原始 PDF 提取的文字，確認是 **PDF 本身的問題**：
- 原始 PDF 中就是 `mn`（沒有乘號）
- 這應該理解為 `m×n`（m 乘以 n）
- 屬於 PDF 轉文字時的正常現象

### 示例位置
```
使用所設計的資料結構，完成維度分別為 mn 的 A 矩陣與 nl 的 B 矩
陣之矩陣相乘（Multiply）運算函式。
```

應理解為：
```
使用所設計的資料結構，完成維度分別為 m×n 的 A 矩陣與 n×l 的 B 矩
陣之矩陣相乘（Multiply）運算函式。
```

### 解決方案
這是 PDF 源文件的問題，不影響理解。閱讀時可根據上下文理解為矩陣維度表示法。
