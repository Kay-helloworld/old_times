## 📚 核心知識速查

### 一、電子商務模式分類

```
【B2B (Business to Business)】企業對企業
範例: 阿里巴巴、台灣經貿網
特色: 大量交易、長期合約

【B2C (Business to Consumer)】企業對消費者  
範例: momo、PChome、Amazon
特色: 零售、品牌直營

【C2C (Consumer to Consumer)】消費者對消費者
範例: 蝦皮、露天拍賣、eBay
特色: 個人賣家、拍賣競標

【C2B (Consumer我 Business)】消費者對企業
├─ 反向拍賣: 消費者出價,企業競標
├─ 網路揪團: 集體議價
└─ 客製化訂製: 消費者指定需求
```

---

### 二、數位行銷關鍵技術

```
【SEO (Search Engine Optimization)】搜尋引擎優化
目標: 提升自然搜尋排名
方法:
├─ 關鍵字優化
├─ 內容品質
├─ 外部連結 (Backlink)
└─ 網站速度

【SEM (Search Engine Marketing)】搜尋引擎行銷  
目標: 付費廣告
方法: Google Ads關鍵字廣告

【社群行銷 (Social Media Marketing)】
平台: Facebook、Instagram、LINE
策略:
├─ 內容行銷 (Content Marketing)
├─ KOL/網紅合作
├─ 社群互動
└─ 精準投放廣告

【長尾效應 (The Long Tail)】
傳統: 20%商品創造80%營收 (暢銷品)
網路: 80%非熱門商品累積可創造可觀營收
      (因庫存成本低、觸及範圍廣)
```

---

### 三、金融科技與支付

```
【行動支付 (Mobile Payment)】
定義: 透過行動裝置進行付款
類型:
├─ NFC (近場通訊): Apple Pay, Google Pay
├─ QR Code掃碼: LINE Pay, 街口支付
└─ App內付款

【第三方支付】
定義: 非銀行的支付中介平台
範例: PayPal, 支付寶, 綠界
功能:
├─ 代收代付
├─ 信任中介 (買賣雙方保障)
└─ 金流整合

【FinTech (金融科技)】
應用:
├─ P2P借貸
├─ 群眾募資
├─ 機器人理財
├─ 區塊鏈金融
└─ 開放銀行 (Open Banking)
```

---

### 四、新興電商概念

```
【共享經濟 (Sharing Economy)】
定義: 共享閒置資源
範例:
├─ Uber/Lyft (共享車輛)
├─ Airbnb (共享房屋)
└─ Gogoro (共享電池)

【社群商務 (Social Commerce)】  
定義: 社群媒體+電子商務
模式:
├─ Facebook/Instagram商店
├─ 直播帶貨
├─ 社群團購
└─ KOL推薦連結

【O2O (Online to Offline)】
定義: 線上導流到線下消費
範例:
├─ 線上訂餐外帶
├─ 線上預約美容
└─ 線上購票實體觀影

【全通路 (Omni-channel)】
整合所有通路提供一致體驗:
線上商店 + 實體門市 + App + 客服
```

## 🎯 重點題型答題技巧

### C2B電子商務三種模式 (112年高考)

```
【網路揪團】
目的: 集體議價,降低單價
平台服務: 湊單、通知、金流
範例: 團購網、集資平台

【個性化訂製】
目的: 個人化需求
平台服務: 設計工具、預覽、生產對接
範例: Nike客製球鞋

【反向拍賣】
目的: 消費者出價,企業競標
平台服務: 需求刊登、廠商競標、撮合
範例: Priceline (飯店)
```

### 網路行銷 vs 傳統行銷 (107年高考)

| 面向 | 傳統行銷 | 網路行銷 |
|------|---------|---------|
| **觸及** | 地理限制 | **全球** |
| **時間** | 營業時間 | **24/7** |
| **成本** | 高 (廣告、通路) | **相對低** |
| **精準度** | 粗略分眾 | **精準投放** |
| **互動** | 單向 | **雙向即時** |
| **成效追蹤** | 困難 | **即時數據** |
| **修正** | 困難 | **即時調整** |
