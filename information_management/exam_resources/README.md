## ⚠️ 資安合併考試特殊處理

### 背景說明
過去一段時間，資訊管理與資訊安全為**合併考試**，科目名稱為「資訊管理與資通安全」。

### 統計資訊
- **合併考試考卷**: 34 份（約44.7%的考卷）
- **合併考試題目**: 138 題（約44.1%的題目）
- **涉及年份**: 104-114年

### 處理原則
1. **保留所有考題**: 所有合併考試的題目都已保留，並標記為資安合併考試
2. **雙重分類**: 複合題型的考題可能同時包含資訊管理與資訊安全關鍵字
3. **分類標準**:
   - 若題目主要涉及資安關鍵字（密碼學、網路安全、漏洞等）→ 標記為「資安類」
   - 若題目主要涉及資訊管理關鍵字（系統開發、專案管理、電子商務等）→ 標記為「資訊管理類」
   - 複合題型可以同時出現在兩個類別中

### 資安相關主題分布（合併考試）

## 🛠️ 使用方法

### 1. PDF轉文字
```bash
cd exam_resources
python3 convert_pdfs_with_pdfplumber.py
```
- 輸入: `original_pdfs/*.pdf`
- 輸出: `processed_text/*.txt`

### 2. 關鍵字提取（jieba）
```bash
python3 extract_all_keywords.py
```
- 使用 jieba 中文分詞 + TF-IDF/TextRank
- 輸出: `analysis_reports/keyword_extraction_report.md`

### 3. 綜合分析（完整關鍵字體系）
```bash
python3 analyze_information_management.py
```
- 基於1200個關鍵字進行分析
- 涵蓋23個領域
- 輸出: `analysis_reports/information_management_comprehensive_analysis.md`
