# 經濟部國營事業聯招（中油、台電、台水、台糖） 資訊類 考試資源

這個目錄專門用來存放並分析「經濟部所屬事業機構新進職員甄試」資訊類科目的歷屆試題。

經濟部聯招的專業科目通常分為：
*   **專業科目A**：計算機概論、網路概論（選擇題）
*   **專業科目B**：資訊管理、程式設計、資料結構、資料庫系統（非選擇題/申論題）

## 目錄結構說明

*   `exam_resources/original_pdfs/`：**請將您下載的經濟部聯招歷屆試題 PDF 檔案放在這裡。**
*   `exam_resources/processed_text/`：系統會將 PDF 轉換成文字檔（Markdown）存放在此，方便進行文字檢索與知識點拆解。
*   `MOC/`：存放由 AI 分析產生的知識點分布報告、重點統計與學習地圖。
